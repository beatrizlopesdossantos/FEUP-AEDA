◆ removeClient()

void Petwalker::removeClient ( string  username)

This function is used to remove a Client of the company. It checks if a client with the same username exists.

  • If that happens that client is erased from the vector, all the scheduled services that he had are also erased and in the services that were already done for that client we substitute it's username for "NoMoreAClient".
  • Else it throws the exception ClientDoesNotExist and catches it displaying the exception message.
    Parameters
    username- Username of the client to remove.

◆ removeEmployee()

void Petwalker::removeEmployee ( string  username)

This function is used to remove a Employee of the company. It checks if a employee with the same username exists.

  • If that happens that employee is erased from the vector, all the scheduled services that he had are also erased (?) and in the services that were already done by that employee we substitute it's username for "NoMoreAEmployee".
  • Else it throws the exception EmployeeDoesNotExist and catches it displaying the exception message.
    Parameters
    username- Username of the employee to remove.
